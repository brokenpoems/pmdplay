# pmdplay
用于 Yamaha YM2203 和 YM2608 的 FM 音源播放器。

支持的文件格式：.M, .M2 以及其他 PMD 支持的格式。

## 编译
请用 Visual Studio 2017 编译，需要下载 [DxLib](http://dxlib.o.oo7.jp).
